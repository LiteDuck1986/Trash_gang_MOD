# Trash_gang_MOD
**Trash_gang_MOD** is a mod for the game **Trash_gang** that adds a in-game cheat menu. It allows players to manipulate game data, inventory, player stats, day timers, and camera settings in real-time.

## Features

The mod menu can be toggled and controlled using the following shortcuts:

| Key | Action |
|-----|--------|
| `F1` | Toggle mod menu visibility |
| `F2` | Switch between tabs |
| `F6` | Start/stop day timer |

### Tabs Overview

#### 1. GameData
- Adjust **Player Speed** and **Movement Level**.
- Modify **Revenue Boost Level**.
- Toggle items like **Trash Magnet** and **Sorting Guide**.
- Manage shop effects and purchases:
  - Reapply all effects.
  - Clear purchased effects.
  - Add $500 to the playerâ€™s money.

#### 2. Inventory
- Adjust **Max Capacity** of inventory.
- Clear or fill items.
- Give the player 10 of the current item in the selected slot.
- Set inventory capacity to 1.

#### 3. Player
- Modify **player speed**.
- Teleport the player to `(0,0,0)`.
- Enable a **noclip** mode (collisions ignored with impassable objects).

#### 4. Day / Timer
- Check if the day timer is running.
- Start or stop the timer.
- Adjust the current time with buttons (`-30s`, `-10s`, `+10s`, `+30s`).

#### 5. Camera
- Adjust **camera zoom** or **FOV**.
- Toggle camera follow player and adjust smoothness. (Removed in the latest update)

## Installation

1. Clone this repository or download the latest release.
2. Simply just launch the game, no injection required and it's Malware free!
3. Launch the game. The mod will initialize automatically after scene load.

## Notes

- The mod is designed for fun only.
- Some features (like noclip) may not work yet. Should be fixed in the next update.
- For player speed, levels and shop effects to work, modify them **before starting the day**.
